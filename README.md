# remove-google-docs-href-tracking-clean-it
Sure! You can install **all required packages at once** with `pip`. For this project, you need:

* `flask` → for the browser UI
* `beautifulsoup4` → for HTML parsing
* `lxml` → optional but recommended for faster and more robust HTML parsing

---

### **Install all at once**

```bash
pip install flask beautifulsoup4 lxml
```

---

### ✅ Optional: Use a `requirements.txt` file

Create a file named `requirements.txt` with:

```
flask
beautifulsoup4
lxml
```

Then install everything with:

```bash
pip install -r requirements.txt
```

This ensures your environment is ready to run the Flask app.
